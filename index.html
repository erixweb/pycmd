Pywarp

Shell para desarolladores web en Windows. Hecha para scrapear, analizar y descargar contenido.

Descargar beta

Hecho para web scraping

Con pywarp puedes hacer web scrapping de forma fácil pasando unos pocos argumentos.

pywarp webscrap https://example.com title -r Example Domain

Guarda todo lo que scrapeas usando el parameto -s (save) en vez de -r (return)

pywarp webscrap https://example.com title -s Scrapped content Path: ./scrapped

Instala recursos rapidamente

Con tan solo la url de un sitio instala su contenido, soporta HTML, otros tipos de texto y instalaciones binarias.

pywarp fetch https://example.com -s Installing resource example.com Installed example.com in 420ms Path: ./installed

Utiliza el parametro -r para solo visualizar el contenido de la página.

pywarp fetch https://example.com -r <html> <main> ... </main> </html>

Comandos por defecto

También se incluyen comandos que vienen por defecto en Windows. Como cd o ls entre otros. Aparte de cd y ls se añade el "vol" para ir a el directorio C:// o E://

pywarp ls - Desktop - Downloads - .dotnet vol E ls - projects - main - work cd work cd mycompany

Comando para VSCode integrado

Para abrir visual studio code en pywarp, puedes utilizar el comando "vscode".

vol E ls - projects - main - work cd work cd mycompany vscode

Comandos npm

Pywarp tiene comandos para inicializar proyectos de node por defecto. Este comando requiere que tengas node.js & npm instalados.

mkdir proyecto create vite --npm ___>./my_project Vanilla JavaScript dev --npm VITE Started vite server at port 5137

Tambien compatibilidad con pnpm utilizando el parametro --pnpm en vez de --npm
