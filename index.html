Pywarp

La herramienta todo en uno para scrapear, analizar, instalar creada en python.

Descargar beta

Hecho para web scraping

Con pywarp puedes hacer web scrapping de forma fácil pasando unos pocos argumentos.

./pywarp pywarp webscrap https://example.com title -r Example Domain

Guarda todo lo que scrapeas usando el parameto -s (save) en vez de -r (return)

./pywarp pywarp webscrap https://example.com title -s Scrapped content Path: ./scrapped

Instala recursos rapidamente

Con tan solo la url de un sitio instala su contenido, soporta HTML, otros tipos de texto y instalaciones binarias.

./pywarp pywarp inst https://example.com -s Installing resource example.com Installed example.com in 420ms Path: ./installed

Utiliza el parametro -r para solo visualizar el contenido de la página.

./pywarp pywarp inst https://example.com -r <html> <main> ... </main> </html>
